"# sentifilm" 
Developed a Deep Learning model using LSTM (Long Short-Term Memory) to
classify 50,000 IMDb movie reviews with 88% test accuracy. Engineered a text
preprocessing pipeline involving tokenization, sequence padding, and vocabulary
indexing of 15,000 words. Implemented a custom prediction interface using
TensorFlow and Keras to provide real-time sentiment analysis and confidence
scores for user-inputted text.
Technologies: Python, TensorFlow, Keras, NumPy, Matplotlib, Google Colab GPU
(Tesla T4).
